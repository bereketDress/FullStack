Employment Management System (EMS)

Full-Stack Web Application |React, Spring Boot, SQL

-Developed a full-stack web application using React for the frontend and Spring Boot for the backend.
-Designed and implemented RESTful APIs to manage employees, roles, and employment records.
-Built responsive and user-friendly UI components using HTML5, CSS3, and JavaScript.
-Integrated backend services with a relational SQL database.
-Applied OOP principles, validation, and error handling to ensure data integrity.
-Conducted functional testing and fixed bugs to improve system reliability and performance.
-Used Git and GitHub for version control and collaborative development.
